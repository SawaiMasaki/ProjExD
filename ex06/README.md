## ブロック崩し（ex06/block.py）
### ゲーム概要
- 実行するとブロックとボールと棒が描画される
- ボールと棒が衝突するとボールが反射する
- ブロックとボールが衝突するとブロックが消滅する
### 操作方法
- 矢印キーで棒を左右に移動する
### 追加機能
- 時間の計測
